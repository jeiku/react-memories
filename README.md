## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jeiku/react-memories.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Add proper variables in '.env'
   ```JS
   CONNECTION_URL = 'enter url'
   ```
4. Start server
   ```sh
   cd server
   npm start
   ```
5. Start client
   ```sh
   cd client
   npm start
   ```
